	**Programa Banco
	*Descrição
	O Programa Banco é um sistema simples de gerenciamento de contas correntes que permite aos 
	usuários realizar operações básicas como depósitos, saques e verificação de saldo.

	*Funcionalidades
	1. Criação de Clientes:

	-Permite registrar informações de novos clientes como CPF, nome, endereço e ano de nascimento.

	2. Criação de Contas Correntes:
	-Permite criar contas correntes associadas a um ou dois titulares.
	-Define um saldo inicial e um limite de crédito total para cada conta.

	3. Depósitos
	-Permite realizar depósitos em contas correntes.
	-Atualiza o saldo da conta corrente após o depósito.

	4. Saques
	-Permite realizar saques de contas correntes.
	-Verifica se há saldo disponível ou se o limite de crédito é suficiente antes de efetuar o saque.

	5.Verificação de Saldo
	-Exibe o saldo atual de uma conta corrente específica.

	6. Resumo da Conta
	-Mostra informações detalhadas sobre uma conta corrente, incluindo 
	 número da conta, titulares, saldo atual, limite atual e status devedor.

	*Uso
	Para executar o programa, compile e execute a classe ProgramaBanco que 
	contém o método main.
